# GDIM32GroupProject

An 8 week project for GDIM 32 at UCI; required utilization of co-op play, AI enemies, AI teammates, SOLID Principles, and design patterns.

## Instructions
Controls:

- Player 1:
  - WASD to move
  - Space to interact
- Player 2:
  - Arrow keys to move
  - Enter/return to interact
